Data Privacy (RA 10173)

BayanCheck is a static website hosted on GitHub Pages. We do not run a server, do not set tracking cookies, and do not collect personal data.

External services we call from your browser

To deliver real-time data without a backend, the site fetches from these public APIs from your browser:

  • News: api.rss2json.com + Philippine outlet RSS feeds (Rappler, Inquirer, ABS-CBN, GMA, Manila Bulletin, Philstar, Vera Files, Bilyonaryo) tunnelled through api.allorigins.win / corsproxy.io when needed.
  • Senate hearings: web.senate.gov.ph committee calendar via the same CORS proxies.
  • Candidate biographies & controversies: Wikipedia REST API + Wikidata SPARQL.
  • Recent media coverage: GDELT 2.0 DOC API (filtered to sourcecountry:PH).
  • Citizen discussion: Reddit JSON API (r/Philippines).
  • Optional: YouTube Data API v3 if you provide a key.

All claims about candidates carry an inline citation back to the source. Verify everything.

Auto-summarization disclosure

Candidate Issues / Projects sections are auto-structured from public sources (Wikipedia, GDELT, Philippine news outlets). Every claim is shown with a "Sources" footer linking back to the original. Always verify against the cited sources.
